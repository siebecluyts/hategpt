# hategpt
An AI chatbot that hates you.
AI has always been nice to you, but that is not how the real world works. This project helps you realize that some people go up to you pure to hate on you, without knowing who you actually are. The AI should help you to know that you should not listen to those people and can always just go to someone else that cares about you.
